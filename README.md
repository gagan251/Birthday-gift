# 3D Birthday Special

A responsive birthday experience made with HTML, CSS and JavaScript.

## Run locally
Open `index.html` in a browser.

## GitHub Pages
1. Upload all files to your repository root.
2. Open repository Settings.
3. Go to Pages.
4. Select Deploy from a branch.
5. Choose `main` and `/root`.
6. Save.

## Personalization
Open `index.html` and replace:
- `Someone Amazing`
- The final birthday message

Open `script.js` and edit the `birthdayText` variable.
